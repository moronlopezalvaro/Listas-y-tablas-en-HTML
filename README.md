# Crear un Horario de Asignaturas usando Listas y Sublistas
Objetivo: Crear un documento HTML que muestre el horario de asignaturas de la semana utilizando listas no ordenadas y ordenadas, con sublistas para mostrar los detalles de cada asignatura (hora y profesor).

# Instrucciones:
Estructura del documento HTML:
Crea un archivo HTML con la estructura básica (DOCTYPE html, html, head, body).
El título de la página debe ser "Horario de Clase".

# Lista no ordenada (Días de la semana):
Dentro del cuerpo (body), crea una lista no ordenada (ul) que contenga cada día de la semana (Lunes, Martes, Miércoles, etc.).

# Sublistas ordenadas (Asignaturas por día con detalles):
Debajo de cada día de la semana, agrega una sublista ordenada (ol) que contenga las asignaturas del día en el orden en que se imparten.
Dentro de cada asignatura, añade otra lista no ordenada (ul) para indicar los detalles de esa asignatura, como la hora y el nombre del profesor.

# Personalización (Opcional):
Puedes agregar más detalles dentro de las sublistas, como la sala de clase o las herramientas necesarias para la asignatura.

# Crear un Horario de Asignaturas usando Tablas

# Instrucciones:
Estructura del documento HTML:
Crea un archivo HTML con la estructura básica.
El título de la página debe ser "Tabla del Horario".

# Crear una tabla para el horario semanal:
Dentro del cuerpo (body), crea una tabla que muestre el horario de asignaturas por días y horas.
La tabla debe tener tantas filas y columnas como precise vuestro horario.
